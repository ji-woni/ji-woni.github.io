# Super Simple Site

A tiny static website (HTML/CSS/JS). No build tools. Drop images in `/images` and edit `index.html`.

## Quick start
1. Edit text in `index.html`. Replace links in the Contact section.
2. Put your images into `/images` and update the `<img src="images/...">` paths + captions.
3. (Optional) Adjust colors/spacing in `styles.css`.

## Publish on GitHub Pages
- Create a public repo named **USERNAME.github.io** (replace USERNAME).
- Upload these files to the repo root.
- In the repo, go to **Settings → Pages** → **Build and deployment** →
  - Source: **Deploy from a branch**
  - Branch: **main**, Folder: **/(root)**
- Visit: https://USERNAME.github.io

## Local preview
Just open `index.html` in your browser.
